Alerts

VISPA-UI includes an expertly-crafted 12 varieties of Alerts which can make your UI building experience effortless.

Basic

To use the alert component you can use the alert class to the alert container.

This is a Basic Alert! 
              
              
<div class="alert">
  <span class="alert__text">
    This is a Basic Alert!
  </span>
</div>

Basic Outline Alert

To use the Basic Outline Primary Alert component you can use the al-primary class to the alert container.

This is a Basic Outline Primary Alert! 
              
              
<div class="alert al-primary">
  <span class="alert__text">
    This is a Basic Outline Primary Alert!
  </span>
</div>

Basic Solid Alert

To use the Basic Solid Primary Alert component you can use the al-solid-primary class to the alert container.

This is a Basic Solid Primary Alert! 
              
              
<div class="alert al-solid-primary">
  <span class="alert__text">
    This is a Basic Solid Primary Alert!
  </span>
</div>

Alert with Icons

To use the Alert with icons component you can use the al-wicon class to the alert container.

This is a Primary Alert With Icons
