Variables

VISPA-UI includes an expertly-crafted Variables which you can customize as per your projects requirements, which can make your UI building experience effortless.

Colors

VISPA-UI has 7 Themed Colors which have the values as given below. To overwrite the color variables just use the same name and give the value as per your requirements in the root of your CSS file. Easy right? So, what are you waiting for? Go now and customize the themed colors. 

              
              
:root {
  --danger:    #FF3860;
  --dark:      #263238;
  --warning:   #FDE047;
  --light:     #E9F3FD;
  --primary:   #23A6F0;
  --secondary: #0F56B3;
  --success:   #48C774;
}
