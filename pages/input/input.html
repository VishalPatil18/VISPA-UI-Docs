Input

VISPA-UI includes an expertly-crafted 8 varieties of Inputs which can make your UI building experience effortless.

Basic Input

To use the Input component give the input class to the container. Copy the below code to use this input component. 

              
              
<div class="input">
  <label class="input__label">Username *</label>	
  <input class="input__field--text" type="text">
</div>

Validation Input

To use the Validation Input component give the inp__error class to the container. Copy the below code to use the validation input component.

Wrong Password! Please Try again. 
              
              
<div class="input inp__error">
  <label class="input__label">Password *</label>
  <input class="input__field--text" type="text">
  <div class="inp__error--msg">
      Wrong Password! Please Try again.
  </div>
</div>

Textbox Input

To use the Textbox Input component give the input__field--text class to the textbox. Copy the below code to use the textbox input component. 

              
              
<div class="input">
  <label class="input__label">Who are you?</label>	
  <textarea class="input__field--text"></textarea>
</div>

Radio Inputs

To use the Radio component give the radio class to the container. Copy the below code to use the Radio input component. 

              
              
<div class="radio">
  <label class="radio__label" for="checked-radio">
    <input 
      class="input__field--radio" 
      id="checked-radio"
      name="checked" 
      type="radio"
      checked
    >
    Checked
  </label>

  <label class="radio__label" for="unchecked-radio">
    <input 
      class="input__field--radio" 
      id="unchecked-radio" 
      name="unchecked" 
      type="radio">
    Unchecked
  </label>

  <label class="radio__label disabled" for="disabled-radio">
    <input 
      class="input__field--radio disabled__input" 
      id="disabled-radio" 
      name="disabled" 
      type="radio" 
      disabled>
    Disabled
  </label>
</div>

Inputs with Icons

To use the Inputs with Icons component give the input__icons class to the container. Copy the below code to use the Inputs with Icons component. 

              
              
<div class="input input__icons">
  <label class="input__label">Email</label>	
  <input 
    class="input__field--text email"
    type="text"
    placeholder="bablu_tailor@gmail.com"
  >

  <label class="input__label">Password</label>	
  <input 
    class="input__field--text password" 
    type="text" 
    placeholder="********"
  >
</div>

Search Input

To use the Search Input component give the search class to the input element. Copy the below code to use the Search Input Component. 

              
              
<div class="input input__icons">
  <input 
    class="input__field--text search"
    type="text" 
    placeholder="Search..."
  >
</div>

Premium Input

To use the Premium Input component give the input__premium class to the container. Copy the below code to use the Premium Inputs Component. 

              
              
<div class="input input__icons input__premium">
  <div class="input__username">
    <input 
      class="input__field--text user" 
      type="text" 
      placeholder="bablu_tailor"
    >
    <label class="input__label">Username</label>
  </div>

  <div class="input__email">
    <input 
      class="input__field--text email" 
      type="text" 
      placeholder="bablu_tailor@gmail.com"
    >
    <label class="input__label">Email</label>	
  </div>

  <div class="input__password">
    <input 
      class="input__field--text password" 
      type="text" 
      placeholder="********"
    >

  </div>
</div>
