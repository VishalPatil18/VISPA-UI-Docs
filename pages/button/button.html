Buttons

VISPA-UI includes an expertly-crafted 18 varieties of Buttons which can make your UI building experience effortless.

Basic Button

To use the Basic Button component you have to add the button class to the button element. 

            
<button class="button">Button</button>

Sized Buttons

Large Button

To use the Large Button component you have to add the btn-lg class to the button. 

            
<button class="button btn-lg">Large</button>

Medium Button

To use the Medium Button component you have to add the btn-md class to the button. 

            
<button class="button btn-md">Large</button>

Small Button

To use the Small Button component you have to add the btn-sm class to the button. 

            
<button class="button btn-sm">Large</button>

Plain Button

To use the Plain Button component you have to add the btn-plain class to the button. 

            
<button class="button btn-plain">Large</button>

Round Button

To use the Round Button component you have to add the btn-round class to the button. 

            
<button class="button btn-round">Large</button>

Action Button

To use the Action Button component you have to add the btn-action class to the button. 

            
<button class="button btn-socialmedia btn-action btn-solid-primary">+</button>

Button With Icon

To use the Button with Icon component you have to add the btn-w-icon class to the button and button__icon class to the icon. 

            
<button class="button btn-w-icon btn-solid-primary">
  <img 
    class="icon-light icon-md button__icon" 
    src="#" 
  >
  W/Icon
</button>
