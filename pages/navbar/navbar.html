Navbars

VISPA-UI includes an expertly-crafted 2 varieties of Navbars which can make your UI building experience effortless.

Basic Navbar

To use the Basic Navbar component copy the code below to use it in your project. 

              
              
<header class="navbar">
  <img 
    class="icon-lg icon-light"
    src="https://raw.githubusercontent.com/VishalPatil18/VISPA-UI/main/assets/linux.svg"
  >
  <span class="navbar__text">TITLE</span>
  <ul class="navbar__list">
    <li class="button btn-plain navbar__btn">Home</li>
    <li class="button btn-plain navbar__btn">About</li>
    <li class="button btn-plain navbar__btn navbar__btn--active">Blogs</li>
    <li class="button btn-plain navbar__btn">Contact</li>
    <li class="button btn-plain navbar__btn">Feedback</li>
  </ul>
  <div class="input input__icons navbar__search">
    <input class="input__field--text search" type="text" placeholder="Search...">
  </div>
</header>
