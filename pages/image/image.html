Images

VISPA-UI includes an expertly-crafted 3 varieties of Image Utility Classes which can make your UI building experience effortless.

Basic Image

To use the Image component give the img class to the Image. The default width and height are 400px.

Round Image

To use the Round Image component give the img-round class to the Image.

Responsive Image

To use the Responsive Image component give the img-res class to the Image. When you increase/decrease the size of the screen the image size also get's decreased using responsive image class.
